## 🚀 Featured Engineering

### 🐄 Podago - Dairy Farm Management System
*A lightweight, highly accessible platform for dairy cooperatives.*
- **Architecture:** Offline-first design prioritizing local caching and seamless syncing.
- **Tech Focus:** Flutter, Firebase, Firestore, Dart.
- **Key Feature:** Purely software and AI-model driven—zero reliance on complex IoT hardware, reducing friction for rural adoption.

### 🌽 MkulimaSmart - Agricultural Marketplace
*Connecting farmers directly to markets with intelligent pricing.*
- **Architecture:** React Native frontend powered by Python-based analytics.
- **Tech Focus:** Linear Regression (price prediction) & Isolation Forest (anomaly detection).
- **Key Feature:** Smart pricing algorithms that analyze real-time market data to prevent exploitation and maximize farmer yields.
